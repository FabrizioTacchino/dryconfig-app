
# DryConfig

Sistema professionale per la configurazione e preventivazione di pareti a secco, contropareti e controsoffitti.

Sviluppato da **Fabrizio Tacchino**.

## Informazioni Progetto

DryConfig è un'applicazione web moderna per professionisti dell'edilizia a secco che permette di:

- Gestire progetti e clienti
- Configurare stratigrafie certificate
- Generare preventivi professionali
- Gestire materiali e coefficienti
- Amministrare utenti e ruoli

## Tecnologie Utilizzate

Questo progetto è costruito con:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
- Le manine di Fabrizio Tacchino

## Come Modificare il Codice

Ci sono diversi modi per modificare questa applicazione.

**Usare un IDE Locale**

Se vuoi lavorare localmente usando il tuo IDE preferito, puoi clonare questo repository e pushare le modifiche.

L'unico requisito è avere Node.js e npm installati - [installa con nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Segui questi passaggi:

```sh
# Passo 1: Clona il repository usando l'URL Git del progetto.
git clone <YOUR_GIT_URL>

# Passo 2: Naviga nella directory del progetto.
cd <YOUR_PROJECT_NAME>

# Passo 3: Installa le dipendenze necessarie.
npm i

# Passo 4: Avvia il server di sviluppo con auto-reload e anteprima istantanea.
npm run dev
```

**Modificare un file direttamente su GitHub**

- Naviga ai file desiderati.
- Clicca il pulsante "Edit" (icona matita) in alto a destra della vista del file.
- Fai le tue modifiche e committa le modifiche.

**Usare GitHub Codespaces**

- Naviga alla pagina principale del tuo repository.
- Clicca sul pulsante "Code" (pulsante verde) vicino in alto a destra.
- Seleziona la scheda "Codespaces".
- Clicca su "New codespace" per lanciare un nuovo ambiente Codespace.
- Modifica i file direttamente nel Codespace e committa e pusha le tue modifiche quando hai finito.
